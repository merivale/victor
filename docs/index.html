The Full Theory

Victor is a model of the English language, thought of as a code for processing messages (structured arrangements of informational choices) into sentences (strings of words). For exposition purposes, the theory is divided into four layers of increasing complexity. If you are new to the theory, start with Plain Messages, and move up from there. For more details, see the README on GitHub.
